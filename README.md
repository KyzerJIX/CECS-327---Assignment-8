## How to run the system

1. Download or clone this GitHub repository.
2. Open the project folder in VS Code or the terminal.
3. Make sure Python is installed.
4. Install psycopg by running:
   pip install psycopg[binary]
5. Open one terminal and run:
   python server.py
6. Enter a port number when asked, for example:
   6500
7. Open a second terminal and run:
   python client.py
8. Connect the client to the same port number.
9. Enter one of the 3 supported queries.
10. Read the server response shown by the client.
11. Type quit to exit the client.
